## Reviewer Run

```bash
git clone <repository-url>
cd <repository-folder>
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
python etl.py
streamlit run app.py
```

No Docker, external database, cloud credentials, API keys, private packages, or manual data preparation are
required. Generated data is intentionally excluded from Git and is created by the ETL command.

## Recommended Review Order

1. Read the root `README.md` and `docs/architecture.md`.
2. Run configuration validation and tests.
3. Run `python etl.py`.
4. Run `streamlit run app.py`.
5. Review pipeline quality and health inside the dashboard.
6. Read `docs/ai-usage.md` and `ai_transcript/transcript.md`.
